# Authoring Skills

Use this skill when you want the AI to **create or update** another skill from instructions in a consistent format.

## When to use

- "Create a skill that does X"
- "Update this skill to follow the standard format"
- "Turn these instructions into a Cursor skill"

## Files

| File | Purpose |
|------|---------|
| **SKILL.md** | Main instructions (create/update process, success criteria) |
| **assets/skill-template.md** | Template structure for new skills |
| **example.md** | Example of using the authoring process |

## Success criteria (from SKILL.md)

- Skill `name` in gerund form: **○○ing-○○**
- `description` in third person, with trigger terms, ≤ 1024 chars
- SKILL.md ≤ 500 lines
- References only one level deep from SKILL.md

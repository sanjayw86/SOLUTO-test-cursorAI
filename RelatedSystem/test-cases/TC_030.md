**Routing profile (Connect):**

After Twilio → Connect, **routing profile** in Connect replaces skill selection in ARCH for queue routing. For **KFS**, use the team-approved **KFS** routing profile (confirm name per environment). **Wrong profile** may land chats on **[つか] Message (Soluto)** instead of **[故] Chat (KFS)**. **App path for this case:** **Mypage → 破損・盗難・紛失 お手続き → チャットでご相談** → **KFS**.

**How to set routing profile:** (1) Open **Connect** for SQA or UAT. (2) **User Management**. (3) Select **User**, search **your name**. (4) Open user → **Routing profile** dropdown → team-approved **KFS** profile → **Save** if required. (5) **ARCH** → **refresh** → verify **Routing Profile**. (6) If wrong, **set the profile again** in Connect (repeat from step 3), refresh ARCH, recheck.

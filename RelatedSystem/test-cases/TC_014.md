## Test Case 14: Call Reservation Reminder SMS is sent (Un-provisioned MDN, AU)

**Test Case ID:** SMS-CALL-REMINDER-AU-001  
**Test Case Title:** Verify Call Reminder SMS is sent and recorded in DB before the actual call reservation time (un-provisioned MDN, service enabled: true, brand AU)  
**Priority:** High  
**Automation ID:** C00558  
**Grade:** D  
**Preconditions:**
- Call reservation SMS conditions: **MDN un-provisioned**, **service enabled: true**, **brand: AU**.
- A call reservation has been made for the test MDN (e.g. via CC tool or Shop UI tool).
- Reminder SMS is sent before 1 hour of the actual call reservation time.
- Access to CC tool / Shop UI tool and ISS DB for verification.

**Test Steps:**

| Step | Action | Expected Result |
|------|--------|-----------------|
| 1 | Have one or more un-provisioned MDNs. If needed, enroll MDN(s) via CSV upload or Orange API (un-provisioned for call reservation, service enabled: true, brand AU). | Un-provisioned MDN(s) are ready. |
| 2 | Go to Soluto Timeslot Web **CC tool** or **Shop UI tool** and make a reservation using one of the test MDNs. | Reservation is created. |
| 3 | Wait until the Reminder SMS is sent (Reminder SMS is sent before 1 hour of the actual call reservation time). | Sufficient time for Reminder SMS to be sent. |
| 4 | In ISS DB, run: `select * from sms_system.message order by created_datetime desc limit 10;` Identify the row for **Reminder SMS** (Source = SUPPORT_SESSION) for the test MDN. | Reminder SMS record is present in ISS DB for the test MDN. |
| 5 | Verify the message text for the Reminder SMS in the DB. | Text matches: **【使い方サポート】　お電話のご予約時間が近づいております。ご予約情報は、専用アプリにてご確認いただけます。専用アプリはこちらからダウンロードしてください。 http://kddi-l.jp/eSd　※なお、ご不在の場合はご予約がキャンセルとなる場合がございます。** (Reminder SMS transmission completed; confirmation via log.) |

**Expected result (overall):** Call Reminder SMS is sent before the reservation slot and verifiable in ISS DB.

**Test Data:**

- **MDN creation requirements (when creating an MDN):**
  - ① "使い方サポート" (usage support) contract  
  - ② Date of birth  
  - ③ Gender  
  - ④ Brand  

- **CC Tool:** https://callcenter.soluto-kddi-sqa.jpnmob-acyan.npr.aws.asurion.net/reservation/search  
- **Shop UI Tool:** https://shop.soluto-kddi-sqa.jp.nonprod-asurion53.com/auShop-entry  

- **Verification queries (備考):**
  - ISS DB (Reminder SMS – Source = SUPPORT_SESSION):  
    `select * from sms_system.message order by created_datetime desc limit 10;`  

**Note:** Reminder SMS is sent before 1 hour of the actual call reservation time. Confirmation is via DB/log. In DB, Source = SUPPORT_SESSION for Reservation-related SMS.

**Postconditions:** Reminder SMS record exists in ISS DB for the test MDN. No mandatory cleanup unless required by project rules.

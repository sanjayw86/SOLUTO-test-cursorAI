## Expected result details

**Pass criteria (each run A/B):**

- 2-line layout is preserved exactly.  
- Multiple spaces in both lines stay visible.  
- Rendering differences are acceptable only if spacing/newline semantics remain preserved.

---

## Expected result (overall)

For **both Soluto and KFS** runs: open chat -> send whitespace-sensitive message -> ARCH closes session -> app re-initiates chat -> prior message appears in history with line breaks and spaces preserved.

**Note:** Use **either** the CSV steps (2–5) **or** the Orange API step (6) for one test run; do not mix unless the scenario requires it. OpenSearch uses **GMT/UTC**; ISS often **JST** — apply project rules when comparing times. SMS to subscribers is typically **11:00–18:00 JST**; after 18:00 JST ISS may show **Scheduled** until the next window, then **Delivered**; OpenSearch may appear at delivered time. Often **two** ATOM API log lines per send; if logs are missing, try index patterns `uap-acyan-api-uat1*`, `uap-acyan-api-uat2*`, clear filters, search by Contract ID or subscriber code, check Titan **carrier_integration.request**, or escalate to FO with subscriber code: [Teams](https://teams.microsoft.com/l/message/19:acdbf9c190e647fbba032efc38a18146@thread.v2/1764928232546?context=%7B%22contextType%22%3A%22chat%22%7D).

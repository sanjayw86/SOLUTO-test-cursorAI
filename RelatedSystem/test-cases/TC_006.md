## Test Case 6: Reminder2 SMS is not sent when service enabled is false (Admin UI – Remind2 ON) (UQ)

**Test Case ID:** SMS-REMIND2-006  
**Test Case Title:** Verify Reminder2 SMS is not scheduled or sent when service enabled is false (Orange API, brand UQ; Admin UI Remind2 ON)  
**Priority:** High  
**Automation ID:** C00640  
**Grade:** D  
**Preconditions:**
- [Admin UI] **SMS Status:** Remind2 ON  
- [Admin UI] **Remind interval:** Remind2 after 1 day  
- OWL/BISCUIT conditions for this test: **service enabled: false**, **brand: UQ**. When service enabled is false, SMS will not be scheduled or delivered; when true, SMS would be scheduled and delivered.
- Orange API registration is enabled for the environment.
- Postman (or equivalent) is available; access to Titan DB and ISS DB for verification queries.

**Test Steps:**

| Step | Action | Expected Result |
|------|--------|-----------------|
| 1 | In Postman, create a new account/MDN that meets the OWL/BISCUIT conditions in advance with **service enabled: false**, **brand: UQ** (see Test Data for MDN requirements). | MDN/account is created with ① "使い方サポート" contract, ② Date of birth, ③ gender, ④ brand UQ, service enabled: false. |
| 2 | Send the POST request to the carrier enrollment API. In the request body, change only **MDN** and **SubscriberCode** as needed for the test. | Request is accepted (2xx); enrollment is registered in the system. |
| 3 | In Titan DB, run: `select * from carrier_integration.message_target mt order by created_datetime desc;` | No SMS is scheduled for the test MDN (no message_target records for Reminder2 for this MDN, or SMS not scheduled for delivery). |
| 4 | In ISS DB, run: `select * from sms_system.message order by created_datetime desc limit 100;` | No Reminder2 SMS (or Welcome/Reminder messages) are present for the test MDN (SMS is not sent when service enabled is false). |
| 5 | In ISS DB (and/or Titan views as per 備考), verify that Reminder2 SMS is not scheduled or sent for that MDN. | Reminder2 SMS is not scheduled and not sent for the test MDN. |

**Expected result (overall):** Reminder2 SMS will not be sent when service enabled is false (brand UQ).

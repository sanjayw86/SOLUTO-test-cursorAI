## Test Case 23: ARCH Soluto chat send/receive — Operator, Connect routing profile, URL, app closure

**Test Case ID:** ARCH-CHAT-SOLUTO-SENDRECV-001  
**Test Case Title:** Verify two-way chat, auto-reply, history after re-entry, adviser URL, and closure with inquiry fields — **Operator** on **[つか] Message (Soluto)**  
**Priority:** High  
**Automation ID:** C00129  
**Grade:** D  

**Role:** **Operator** (must exchange chats; not Viewer / Business admin–only).

**Preconditions:**

- **User:** **au Tsukasapo** test user; **chat is not already open** for this session.  
- **Data:** IRIS (or dependent) data created using the **Stub** tool, per project procedure.  
- **Agent:** **Operator** account; **routing profile** set per **Routing profile** subsection below.  
- **ARCH:** **Soluto** message context (**[つか] Message (Soluto)**), not **KFS**, for this test case.  
- **Agent status (Amazon Connect):** **受付可 (Available)** — required to receive chats (*per internal ARCH doc*).  
- **Browser:** **Chrome** for ARCH.  
- **Environment alignment:** Use **one** line end-to-end (e.g. **SQA1** app + **SQA1** ARCH + **SQA** Connect). Do not mix SQA app with UAT ARCH.

**Routing profile (Connect):**

After Twilio → Connect, **routing profile** in Connect replaces skill selection in ARCH for queue routing. For **Soluto**, use e.g. **SOLUTOGeneralRoutingProfile** (confirm with team). **Wrong profile** may land chats on **[故] Chat (KFS)**. App path for this case: **困った・使い方がわからない → メッセージでご相談** → **Soluto**.

**How to set routing profile:** (1) Open **Connect** for SQA or UAT. (2) **User Management**. (3) Select **User**, search **your name**. (4) Open user → **Routing profile** dropdown → e.g. **SOLUTOGeneralRoutingProfile** → **Save** if required. (5) **ARCH** → **refresh** → verify **Routing Profile**. (6) If wrong, **set the profile again** in Connect (repeat from step 3), refresh ARCH, recheck.

**Test Steps:**

### Phase A — Connect routing profile + ARCH readiness

| Step | Action | Expected Result |
|------|--------|-----------------|
| 1 | Open **ARCH** for **SQA** or **UAT** (match app). Select **Skills** as **"[つか] Message (Soluto)"** or **"[故] Chat (KFS)"** per requirement (*this case: **Soluto***). Sign in as **Operator** when prompted. | Correct env + skill; **Operator** logged in (not Viewer-only). |
| 2 | On ARCH, view **Routing Profile** vs requirement for step 1 (e.g. **SOLUTOGeneralRoutingProfile** — confirm with team). | Profile visible and assessed. |

# Manual Test Cases: DataBase / Denodo — `bookmark_by_button` event log (`report_vfaq_analytics_report_realtime`)

**Source:** Tsukasapo app search flow + Denodo realtime analytics report  
**Viewpoint:** Event logging (`metadata_eventname`)  
**Purpose:** Verify `bookmark_by_button` is logged when user bookmarks article using the `Save article` button.

---

## Test Case 39: Verify `bookmark_by_button` event value in Denodo report

**Test Case ID:** DB-DENODO-BOOKMARK-BY-BUTTON-001  
**Test Case Title:** Verify that `bookmark_by_button` value is displayed in event log when user bookmarks content with `Save article` button in `report_vfaq_analytics_report_realtime`  
**Priority:** High  
**Automation ID:** C00645a  
**Grade:** C

**Preconditions:**
- App installed.
- Provisioning done.
- Tsukasapo user account available.
- Access to Denodo app/report `report_vfaq_analytics_report_realtime`.

**Test Steps:**

| Step | Action | Expected Result |
|------|--------|-----------------|
| 1 | Launch app. | App is launched. |
| 2 | Tap **Search** on footer. | Search screen opens. |
| 3 | Tap `困った・使い方がわからない` tile. | Device series bubble page is displayed. |
| 4 | Tap any Android series (for example, XPERIA). | One device series is selected. |
| 5 | Select one option from available bubbles. | FAQ option is selected and related content list is shown. |
| 6 | Open one available content item. | FAQ content opens. |
| 7 | Bookmark content by tapping `Save article` button. | User bookmarks content with button action. |
| 8 | Open Denodo app and search report `report_vfaq_analytics_report_realtime`. | Report is displayed successfully. |
| 9 | Validate column `metadata_eventname`. | `bookmark_by_button` is displayed under `metadata_eventname`. |

**Test Data:**
- Brand/user: Tsukasapo provisioned user.
- Device path: Any Android series (example XPERIA).
- Report: `report_vfaq_analytics_report_realtime`.

**Denodo query (realtime):**
```sql
SELECT *
FROM enterprise_acyan.report_vfaq_analytics_report_realtime
where metadata_eventname like '%bookmark_%'
order by collectiontime desc
limit 100;
```

**Denodo query (historic, next day):**
```sql
SELECT *
FROM enterprise_acyan.report_vfaq_analytics_report_historic
where metadata_eventname like '%bookmark_%'
order by collectiontime desc
limit 100;
```

**Postconditions:**
- Evidence captured for app bookmark action and Denodo report row.

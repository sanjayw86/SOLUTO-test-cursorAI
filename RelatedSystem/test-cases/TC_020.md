## Test Case 20: Call Reservation, Reminder, and Cancellation SMS in ISS and Horizon SIGMA MEMO logs in OpenSearch (Un-provisioned MDN, AU)

**Test Case ID:** SMS-SIGMA-CALL-HORIZON-001  
**Test Case Title:** Verify Call Reservation SMS, Call Reminder SMS, and Call Cancellation SMS in ISS DB; verify Horizon SIGMA MEMO logs in OpenSearch for each (`/carrierinterface/v1/sigma/memo`)  
**Priority:** High  
**Automation ID:** C00654  
**Grade:** D  

**Preconditions:**

- Call SMS conditions: **MDN un-provisioned**, **service enabled: true**, **brand: AU** (adjust if testing UQ).
- Test MDN available (enrolled via CSV or Orange API so the line remains un-provisioned for call reservation, per project rules).
- Access to Soluto Timeslot Web **CC tool** and/or **Shop UI tool**, **ISS DB**, **ISS OpenSearch Dashboard**, **Titan DB** (troubleshooting).
- MDN / contract when creating: ① "使い方サポート" contract, ② Date of birth, ③ Gender, ④ **Brand AU** (adjust if testing UQ).
- When creating the reservation, choose a date/time such that **Call Reminder SMS** can be sent (**before 1 hour** of the actual call reservation time) **before** you cancel — otherwise reminder steps cannot be verified.
- Know the test **Customer code** / subscriber for the MDN (to match Horizon log payload).

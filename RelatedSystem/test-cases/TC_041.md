**Reference SQL template** (replace `<PROJECT_DATASET_TABLE>` with the full table from the console, e.g. intraday table for today; adjust `model` and `scope` per run):

```sql
SELECT CAST(FORMAT_TIMESTAMP('%Y-%m-%d %H:%M:%S', TIMESTAMP_MICROS(event_timestamp), 'Asia/Tokyo') AS DATETIME) AS jst,
  param_value,
  *
FROM (
  SELECT *,
    (SELECT value.string_value FROM UNNEST(event_params) WHERE key = 'model') AS model,
    (SELECT value.string_value FROM UNNEST(event_params) WHERE key = 'action') AS action,
    (SELECT value.string_value FROM UNNEST(event_params) WHERE key = 'scope') AS scope,
    (SELECT value.string_value FROM UNNEST(event_params) WHERE key = 'param_value') AS param_value
  FROM `<PROJECT_DATASET_TABLE>`
)
WHERE event_name = 'logger_debug'
  AND action = 'ScreenViewed'
  AND scope = 'splash'
  AND model = 'Pixel 5'
ORDER BY event_timestamp DESC;
```

**Note:** Example `model = 'Pixel 5'` is illustrative — use your device’s exact model string. For **iOS**, use the exact iPhone model from NMS / team reference.
